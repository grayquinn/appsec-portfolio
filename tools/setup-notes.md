# Tools Setup Notes

## Burp Suite Community
- Download: portswigger.net/burp/communitydownload
- Free version — intercept, modify, replay HTTP requests
- Primary tool for web app security testing

## Semgrep
- Download: semgrep.dev
- Free SAST tool — scans C# .NET code for vulnerabilities
- Run against your own codebase

## OWASP ZAP
- Download: zaproxy.org
- Free DAST tool — dynamic scanning against running apps

## OWASP WebGoat
- docker run -p 8080:8080 webgoat/webgoat
- Deliberately vulnerable app — safe practice target
